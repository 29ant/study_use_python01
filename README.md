# 英英在线词典解析平台

## 项目简介

该英英在线词典解析平台主要基于python编写，数据库为mysql.

## 系统说明

用户可以登陆和注册、添加历史记录、查询历史记录、查单词。

### 普通用户使用

英英在线词典解析平台，Windox操作系统也能运行，但是Linux系统运行，效果会更好，平台中的某些模块，Windox操作系统，不支持。

把所需单词导入到数据库时，只需运行insert_word文件。由于单词很多，需要导入一段时间，如果有更快的导入方式，请联系我。
